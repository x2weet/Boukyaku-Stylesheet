忘却スタイルシート・取説

概要

X2ODPで用ゐてゐるスタイルシートです。

特徴

  • シンプルなスタイル。
  • ユーザスタイルシートとしても使へる。
  • 「あつさり」指向。

使用方法

HTML 4.01 Strict DTDに妥当な文書に適用する場合、例へばこの様に:

<link rel="stylesheet" type="text/css" href="boukyaku.css">

その他関連規格等

「IPA Pゴシック」、「ＭＳ Ｐゴシック」及びLatin Modernフォントファミリの「LMMono10」・「LMMono12」を指定してゐる部分が有ります。此等がインストールされてゐればスタイリングに利用されると思ひます。

ログのやうなもの

  1. スタイルシート「トントン引力」から派生した、スタイルシート「亡骸スタイルシート」から、派生。
  2. 適宜改良中。
